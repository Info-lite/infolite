Ｌ行Ｍ列分割表の検定

練習問題1

都内のある学校で、修学旅行先の希望調査を行いました。クラスは、Ａ、Ｂ、Ｃの3クラスあります。調査結果を整理したのが、次の3行4列分割表です。各クラスの希望の傾向は同じといえるでしょうか。検定してください。

検定の手順

Ｌ行Ｍ列分割表の検定には、2行2列分割表の検定と同様にピアソンの$latex \chi^2$検定を用いることができます。

仮説の設定

  • 帰無仮説H0：修学旅行先の希望の傾向はクラスによらず同じである
  • 対立仮説H1：修学旅行先の希望の傾向はクラスによって異なる

有意水準αの設定

有意水準α = 0.05とします。

検定統計量$latex \chi^2$値の算出

今回、検定統計量$latex \chi^2$は以下の数式で求まります。 $latex \displaystyle \chi^2 = \sum_{i=1}^L \sum_{j=1}^M \frac{{\left( f_{ij} - t_{ij} \right)}^2}{t_{ij}}$ ここで、$latex f_{ij}$は以下の表のとおり、Ｌ行Ｍ列分割表における$latex i$行$latex j$列の度数です。一方、$latex t_{ij}$は$latex f_{ij}$に対する期待値で、 $latex \displaystyle t_{ij} = \frac{l_i m_j}{N}$ で与えられます。なお、L = 2、M = 2を代入し、展開すると2行2列分割表の場合の式になります。
Ｌ行Ｍ列分割表
項目B1 項目B2 … 項目Bj … 項目BM-1 項目BM 計
項目A1 $latex f_{11}$ $latex f_{12}$ … $latex f_{1j}$ … $latex f_{1M-1}$ $latex f_{1M}$ $latex l_1$
項目A2 $latex f_{21}$ $latex f_{22}$ … $latex f_{2j}$ … $latex f_{2M-1}$ $latex f_{2M}$ $latex l_2$
⋮ ⋮ ⋮ ⋱ ⋮ ⋱ ⋮ ⋮ ⋮
項目Ai $latex f_{i1}$ $latex f_{i2}$ … $latex f_{ij}$ … $latex f_{iM-1}$ $latex f_{iM}$ $latex l_i$
⋮ ⋮ ⋮ ⋱ ⋮ ⋱ ⋮ ⋮ ⋮
項目AL-1 $latex f_{L-11}$ $latex f_{L-12}$ … $latex f_{L-1j}$ … $latex f_{L-1M-1}$ $latex f_{L-1M}$ $latex l_{L-1}$
項目AL $latex f_{L1}$ $latex f_{L2}$ … $latex f_{Lj}$ … $latex f_{LM-1}$ $latex f_{LM}$ $latex l_L$
計 $latex m_1$ $latex m_2$ … $latex m_j$ … $latex m_{M-1}$ $latex m_M$ $latex N$

自由度の算出

また、検定統計量$latex \chi^2$は自由度φの$latex \chi^2$分布に従うことがわかっています。自由度φは以下の式で求まります。 φ = (L - 1)×(M - 1) 2行2列分割表の場合、検定統計量$latex \chi^2$は自由度1の$latex \chi^2$分布に近似的に従うことがわかっていますが、これも上式にL = 2、M = 2を代入した結果に一致します。

p値の算出

有意水準と比較するp値を求めます。p値は、自由度φの$latex \chi^2$分布において、検定統計量$latex \chi^2$値より大きい値が発生する確率です。

判定

自由度φ = 6の$latex \chi^2$分布はこのようなグラフを描きます。このグラフで、p値を表す面積が有意水準の確率を表す面積より大きいか小さいかで帰無仮説の棄却を決定します。
p値 ≤ 有意水準α
帰無仮説H0を棄却する
p値 > 有意水準α
帰無仮説H0を受容する

Excelの操作

Excelを使って、練習問題1に取り掛かりましょう。
  1. 表1のデータをExcelに入力します。
  2. 今回は、2行2列分割表を作成する操作では分割表をうまく作成できません。そこで、ピボットテーブル機能を使って、分割表を手動で作成します。[挿入]タブをクリックし、「ピボットテーブル」をクリックします。
  3. 「ピボットテーブルの作成」ウィンドウが開いたらOKをクリックし、ピボットテーブル用のシートを作成します。
  4. ピボットテーブル用シートの右端の「ピボットテーブルのフィールド」で、クラスをドラッグして行へドロップします。
  5. 同様に、修学旅行先を列へ、人数を値へドラッグドロップし、分割表を完成させます。
  6. 検定のために期待値用の項目を入力します。
  7. 期待値用のデータを入力します。
    • "B12"：=$F5*B$8/$F$8
  8. オートフィルで他のセルにもデータを入力します。
  9. 念のため行周辺度数も求めます。
    • "F12"：=SUM(B12:E12)
  10. オートフィルで他のセルにもデータを入力します。
  11. 同様に列周辺度数も求めます。
    • "B15"：=SUM(B12:B14)
  12. 検定のために偏差用の項目を入力します。
  13. 偏差用のデータを入力します。
    • "B19"：=B5-B12
  14. 平方偏差を期待値で割った調整平方偏差用の項目を入力します。
  15. 調整平方偏差用のデータを入力します。
    • "B26"：=B19^2/B12
  16. 検定用の項目を入力し、既知のデータを入力します。
  17. 検定用のデータを算出するために、以下のように入力します。
    • "I3"：=SUM(B26:E28)（検定統計量$latex \chi^2$値）
    • "I4"：=(COUNTA(A5:A7)-1)*(COUNTA(B4:E4)-1)（自由度φ）
    • "I5"：=CHISQ.DIST.RT(I3,I4)（p値）
なお、次々項で偏差について考察するために、ここでは偏差用のデータを求め、それに基づいて検定していますが、"I5"セルはCHISQ.TEST関数を使って度数と期待値とから直接=CHISQ.TEST(B5:E7,B12:E14)と求めることも可能です。

結果

p値 = 2.51009×10-09が求まりました。下図の塗りつぶされた領域が全体に対してpの割合になっています。 p値 = 2.51009×10-09 < 有意水準α = 0.05 なので、帰無仮説H0は棄却されます。 よって、修学旅行先の希望の傾向はクラスによって異なるといえます。

論文では

論文では以下のような文章になります。
表1に関して、$latex \chi^2$検定を行ったところ、有意差が認められた（$latex \chi^2$(6) = 51.35790831, p < .05）。よって、修学旅行先の希望の傾向はクラスによって異なるといえる。

Ｌ行Ｍ列分割表の偏差（残差）のチェック

度数と期待値との差である偏差（残差）を見てみましょう。残差の絶対値が大きいところが特徴的であるといえます。 残差を見てみると、Ａクラスは北海道・東北が10.96178344と高く、Ｂクラスは東海・北陸が14.77070064、Ｃクラスは四国・九州が11.80254777とそれぞれ高い数値を出しています。このことから、Ａクラスは北海道・東北、Ｂクラスは東海・北陸、Ｃクラスは四国・九州に行きたいという傾向があることがわかりました。 このような分析を残差分析と呼びます。

Ｌ行Ｍ列分割表のグラフ表現

練習問題1のグラフを作成します。

Excelの操作

練習問題1のファイルを使います。
  1. "A4"から"E7"を選択します。
  2. [挿入]タブをクリックし、「おすすめグラフ」をクリックします。
  3. 「グラフの挿入」ウィンドウが開いたらOKボタンをクリックします。
  4. 希望先のグラフがクラスごとに作成されました。
グラフにすることで各項目ごとのパターンの違いを把握できます。ここでも、Ａクラスは北海道・東北、Ｂクラスは東海・北陸、Ｃクラスは四国・九州が多いということがわかりました。 なお、以上の手順はグラフ作成の一般的な方法になっています。ピボットテーブルに対しては、ピボットテーブルのどこかをクリックすることで、下図のように現れる[ピボットテーブル ツール]の[分析]タブの「ピボットグラフ」をクリックする方法があります。それによって、「グラフの挿入」ウィンドウが開き、ピボットテーブルを適切に視覚化したグラフを作成することができます。
