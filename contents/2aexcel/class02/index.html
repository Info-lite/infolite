母平均に関する検定と推定

日頃私たちが得られるデータは、対象の特性をもれなく測定したデータである母集団（population）そのものではなくて、母集団の中からいくつか抜き出してきたデータである標本（sample）であることが一般的です。私たちが実際にできることは、その標本をもとに母集団を推測することです。 本講では、標本に基づいて算出した平均（標本平均、sample mean）から母集団の平均（母平均）を検定したり、推定したりする方法を学んでいきましょう。

1つの母平均に関する検定（母標準偏差が既知のとき）

練習問題1

複数店舗を展開しているあるファストフード店では、注文を受けてから商品を出すまでに平均60秒かかっているとします。 そこで、いくつかの店舗で何人かの新入店員たちに対して同様の時間を計測したところ、以下のデータを得ました。今年の新入店員全体では注文を受けてから商品を出すまでに平均60秒かかると判断して良いでしょうか。なお、商品を出すまでの標準偏差の値は10秒で、店舗や店員が代わってもこの値は変わらないものとします。

仮説検定の流れ

練習問題1を例にあげると、今年の新入店員全体は注文を受けてから商品を出すまでに平均60秒かかる、すなわち、通常の商品提供平均時間と今年の新入店員たちの商品提供平均時間とが等しい、と判断して良いかを考えるための仮説検定（hypothesis testing）は、
  1. “通常”の商品提供平均時間と“新入店員”全員の商品提供平均時間とが「等しい」という帰無仮説H0と、両者が「等しくない」という対立仮説H1とを考えます。
  2. 平均と標準偏差などから検定統計量$latex z$を求めます。
  3. 求めた検定統計量が起こり得る確率（p値）を求めます。
  4. 有意水準$latex \alpha$（アルファ）とp値とを比較して、どちらの仮説を採択するか決定します。
の流れになります。 帰無仮説（null hypothesis）は極端に言えば、棄てる前提の、つまり、否定したい仮説のことです。先ほどの流れの説明では練習問題1の帰無仮説として、通常の商品提供平均時間と新入店員全員の商品提供平均時間とが「等しい」と設定されています。この仮説を退けるためには、練習問題1の場合、「等しい」状況において商品提供時間の平均が表1に示されたような(64 + 61 + 67 + 57 + 62 + 53 + 69 + 49 + 73 + 59 + 67 + 50 + 58 + 62 + 56 + 61) ÷ 16 = 60.5になることは稀である、すなわち、めったに起こらないことを示す必要があります。つまり、その状況が起こり得る確率p値が「ある小さな値（有意水準$latex \alpha$）」以下であることを示す必要があります。これが示されれば帰無仮説は棄却され、主張したい内容である対立仮説（alternative hypothesis）が採択されます。ここで、対立仮説は先ほどの流れの説明では「等しくない」と設定されています。一方、p値が$latex \alpha$より大きい場合、練習問題1の「等しい」状況で商品提供時間の平均が60.5になることはよくあることになり、帰無仮説を受容することになります。

仮説の設定

ここでは練習問題1に対して、
  • 帰無仮説H0：$latex \mu = \mu_0$
  • 対立仮説H1：$latex \mu \ne \mu_0$
と設定します。ここで、$latex \mu$（ミュー）は今年の新入店員全員の商品提供時間の平均を表し、$latex \mu_0$は全店舗・全店員の商品提供時間の平均を表します。すなわち、$latex \mu$は今年の新入店員全員分の商品提供時間を母集団とする母平均で未知、その母集団の一部が標本で表1に示す新入店員何人かの商品提供時間となります。一方、$latex \mu_0$は全店舗・全店員の商品提供時間を母集団とする母平均となり、その値は60と既知です。 この帰無仮説は言い換えると、表1に示す$latex N$ = 16の商品提供時間は、母平均$latex \mu_0$が60である母集団、すなわち全店舗・全店員の商品提供時間という母集団からランダムに抽出された標本であると仮定していることになります。

有意水準$latex \alpha$の設定

帰無仮説を棄てるためには、その状況がめったに起こらないことを示す必要がありますが、「めったに」の程度は有意水準（significance level）$latex \alpha$で表され、その値は経験的に0.05（5%）か0.01（1%）を用います。後者のほうが、より厳しい条件となります。今回の検定では、有意水準$latex \alpha$ = 0.05とします。詳しくは有意水準を参照してください。

検定統計量$latex z$値の算出（正規分布として考える）

$latex z$は以下の数式で求まります。 $latex \displaystyle z = \frac{\Bar{x} - \mu_0}{\frac{\sigma}{\sqrt{N}}}$ ここで、$latex \Bar{x}$は標本平均でその値は表1に示されている標本から算出可能、$latex \mu_0$は全店舗・全店員の商品提供時間を母集団とする母平均でその値は60と既知、$latex \sigma$（シグマ）は全店舗・全店員の商品提供時間を母集団とする母標準偏差でその値は10と既知、$latex N$は標本の大きさでその値は16と既知です。 なお、ここでは、各店舗・各店員の商品提供時間の分布が平均60、標準偏差10の分布である、言い換えると、各店舗・各店員の商品提供時間が平均60、標準偏差10の分布に従うと考えています。帰無仮説、つまり、表1に示す$latex N$ = 16の商品提供時間は、母平均$latex \mu_0$が60である母集団、すなわち全店舗・全店員の商品提供時間という母集団からランダムに抽出された標本であるが成立する場合、標本平均$latex \Bar{x}$は平均$latex \mu_0$、標準偏差$latex \sigma / \sqrt{N}$（標準誤差）の正規分布に従うことがわかっています。これに基づいて、標本平均60.5が帰無仮説下で生じ得る確率p値を考えることができるようになっています。 ただし、正規分布に従うといっても、正規分布の平均や標準偏差は様々です。そこで、上式のように、平均$latex \mu_0$、標準偏差$latex \sigma / \sqrt{N}$の正規分布に従う値$latex \Bar{x}$を、平均0、標準偏差1の標準正規分布（standard normal distribution）に従う値に変換する、標準化（standardization）と呼ばれる処理が必要となります。標準化によって、平均や標準偏差がそれぞれ異なる正規分布に従うデータも、標準正規分布に従う共通の検定統計量（test statistic）$latex z$値へ変換することができ、その結果、問題が変わっても対象が変わっても、帰無仮説が成立する状況下で標本平均がある値（練習問題1では60.5）になる確率を、検定統計量$latex z$値が生じ得る確率p値として、同じように求めることができるようになっています。

p値の算出

有意水準$latex \alpha$と比較する確率p値を算出します。p値は、標準正規分布において$latex -z$以下の値が発生する確率と$latex z$以上の値が発生する確率の和です。標本の取り方によって標本平均$latex \Bar{x}$の値は大きくなったり小さくなったり変動しますが、帰無仮説の下では$latex \Bar{x}$と$latex \mu_0$との差は小さいので、検定統計量$latex z$も0に近い値になり、p値は大きくなります。逆に、帰無仮説が成立しない場合、$latex z$は極端に大きいか、極端に小さい（負の数になる）ため、p値は小さくなります。

判定

標準正規分布はこのようなグラフを描きます。このグラフで、p値を表す面積が有意水準の確率を表す面積より大きいか小さいかで帰無仮説の受容か棄却かを決定します。
p値 ≤ 有意水準$latex \alpha$
帰無仮説H0を棄却する
p値 > 有意水準$latex \alpha$
帰無仮説H0を受容する

Excelの操作

ここまで検定の理論について説明してきましたが、Excelを用いて練習問題1に取り掛かりましょう。
  1. 表1のデータをExcelに入力します。
  2. 検定用の項目を入力し、既知のデータを入力します。
  3. 検定用のデータを算出するために、以下のように入力します。
    • "D4"：=COUNTA(A2:A17)（標本の大きさ）
    • "D5"：=AVERAGE(A2:A17)（標本平均）
    • "D6"：=D3/SQRT(D4)（標準誤差）
    • "D7"：=STANDARDIZE(D5,D2,D6)（標準化）
    • "D8"：=(1-NORM.S.DIST(D7,TRUE))*2（p値）
    ここで、STANDARDIZE($latex \Bar{x}$,$latex \mu_0$,$latex \sigma / \sqrt{N}$)は、平均$latex \mu_0$、標準偏差$latex \sigma / \sqrt{N}$の正規分布に従う$latex \Bar{x}$を標準化し、$latex z$を求めます（STANDARDIZE関数）。また、NORM.S.DIST($latex z$,TRUE)は標準正規分布において$latex z$未満の値が発生する確率を表すので、1-NORM.S.DIST($latex z$,TRUE)によって、標準正規分布において$latex z$以上の値が発生する確率を求めています（NORM.S.DIST関数）。標準正規分布は左右対称な分布なので、標準正規分布において$latex -z$以下の値が発生する確率は標準正規分布において$latex z$以上の値が発生する確率と等しく、したがって(1-NORM.S.DIST($latex z$,TRUE))*2でp値を求めることができます。
なお、"D8"は、表1のデータと既知のデータだけを用いて、=2*MIN(Z.TEST(A2:A17,D2,D3),1-Z.TEST(A2:A17,D2,D3))と求めることもできます。Z.TEST(データ,$latex \mu_0$,$latex \sigma$)は標準正規分布において$latex z$未満の値が発生する確率を表し、1-Z.TEST(データ,$latex \mu_0$,$latex \sigma$)は標準正規分布において$latex z$以上の値が発生する確率を表します（Z.TEST関数）。それらの確率で小さいほうが必要な確率なので、MIN(Z.TEST(データ,$latex \mu_0$,$latex \sigma$),1-Z.TEST(データ,$latex \mu_0$,$latex \sigma$))で小さいほうの確率を選択します。標準正規分布は左右対称な分布なので、標準正規分布において$latex -z$以下の値が発生する確率は標準正規分布において$latex z$以上の値が発生する確率と等しく、2*MIN(Z.TEST(データ,$latex \mu_0$,$latex \sigma$),1-Z.TEST(データ,$latex \mu_0$,$latex \sigma$))でp値を求めることができます。

練習問題2

文部科学省平成20年度学校保健統計調査の結果によると、6歳男児の身長は全国平均（仮説平均$latex \mu_0$）が 116.7 cmであることが分かっています。一方、同時期のある地域の6歳男児16名の身長は以下のとおりでした。この地域の6歳男児の身長は全国平均よりも高いと言えるでしょうか。

仮説の設定

  • 帰無仮説H0：$latex \mu = \mu_0$
  • 対立仮説H1：$latex \mu \ne \mu_0$
ここで、$latex \mu$はこの地域の6歳男児全員の平均身長、すなわち、この地域の6歳男児全員の身長を母集団とする母平均です。

有意水準$latex \alpha$の設定

有意水準$latex \alpha$ = 0.05 とします。

検定統計量$latex t$値の算出

検定統計量$latex t$は以下の数式で求まります。 $latex \displaystyle t = \frac{\Bar{x} - \mu_0}{\frac{s}{\sqrt{N}}}$ ここで$latex \Bar{x}$は標本平均、$latex \mu_0$は6歳男児の全国平均身長、$latex s$は標本の標準偏差、$latex N$は標本の大きさです。 今回は、練習問題1と違って母標準偏差が未知です。このような場合は、母標準偏差を標本の標準偏差$latex s$で代用して$latex \Bar{x}$を標準化するため、標準化された値（標準得点、standard score）は標準正規分布ではなくt分布に従います。t分布（Student's t-distribution）は、標準正規分布同様の釣鐘型をしていますが、自由度$latex \phi$（ファイ）によって形状が変わります。なお、この検定においてt分布における自由度は標本の大きさ$latex N$を用いて$latex \phi = N - 1$で与えられ、$latex \phi$が30以上で標準正規分布とほぼ同一の形となります。

p値の算出

有意水準$latex \alpha$と比較する確率p値を計算します。p値は、自由度$latex \phi$のt分布において、$latex -t$未満の値が発生する確率と$latex t$より大きい値が発生する確率の和です。

判定

t分布はこのようなグラフを描きます。 このグラフで、p値を表す面積が有意水準の確率を表す面積より大きいか小さいかで帰無仮説の棄却を決定します。なお、図のとおり、自由度$latex \phi$によって、このt分布のグラフは変わります。
p値 ≤ 有意水準$latex \alpha$
帰無仮説H0を棄却する
p値 > 有意水準$latex \alpha$

結果

最後に、標本に基づいて求めた標本平均などの統計量（statistic）から母平均を推定する方法を学びましょう。

区間推定

区間推定（interval estimation）は、母平均や母分散などの母数（parameter）を1つの値として推定（点推定）するのではなく区間として推定します。すなわち、推定したい母平均や母分散などを含んでいる範囲はどこからどこまでなのかを考えます。例えば、次式は、体重の母平均の取り得る範囲（信頼区間）を示しています。 57 kg ≤ $latex \mu$ ≤ 67 kg （信頼係数95%） → この式が成立する確率は95%です。 母平均$latex \mu$の(1 - $latex \alpha$) × 100%信頼区間： $latex \Bar{x} - t_\phi (\alpha) \frac{s}{\sqrt{N}} < \mu < \Bar{x} + t_\phi (\alpha) \frac{s}{\sqrt{N}}$ ここで、$latex \Bar{x}$は標本平均、$latex s$は標本の標準偏差、$latex \phi = N - 1$は自由度、$latex N$は標本の大きさ、$latex t_\phi (\alpha)$は自由度$latex \phi$のt分布において$latex -t$未満の値が発生する確率と$latex t$より大きい値が発生する確率との和が$latex \alpha$となるような$latex t$を表します。なお、(1 - $latex \alpha$) × 100%を信頼係数（confidence coefficient）と呼び、$latex t_\phi (\alpha)$を自由度$latex \phi$のt分布における両側確率$latex \alpha$の$latex t$の臨界値（critical value）と呼びます。また、$latex \Bar{x} - t_\phi (\alpha) s / \sqrt{N}$を下側信頼限界（lower confidence limit）、$latex \Bar{x} + t_\phi (\alpha) s / \sqrt{N}$を上側信頼限界（upper confidence limit）と呼び、下側信頼限界から上側信頼限界までの区間を信頼区間（confidence interval）と呼びます。

"G4"は=D4-CONFIDENCE.T(1-G2,D5,D3)、"G5"は=D4+CONFIDENCE.T(1-G2,D5,D3)、と求めることもできます（ただし、"G2"の表示形式を「標準」に変えておく必要があります）。CONFIDENCE.T($latex \alpha$,$latex s$,$latex N$)は$latex t_\phi (\alpha) s / \sqrt{N}$を与えます（CONFIDENCE.T関数）。
