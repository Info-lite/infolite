回帰分析

変数の関係を探る流れ

相関分析では、2変数の相関関係がわかりました。この関係を式で表すことで、一方の変数から他方の変数を予測したりすることができます。この式を求めることなどを回帰分析（regression analysis）と呼び、求められた式を回帰式（regression equation）と呼びます。 回帰分析では、例えば、$latex x$、$latex y$の2変数について、$latex x$を説明変数、$latex y$を目的変数として、その関係を表す式（$latex y$の$latex x$への回帰式（regression equation of $latex y$ on $latex x$））を求めます。なお、回帰式は$latex x$と$latex y$の関係を完全には表しきれていない（$latex x$から$latex y$の予測の精度が高くない）場合もあることに注意してください。
回帰分析の分類と回帰式の例
線形回帰（linear regression） 非線形回帰（nonlinear regression）
単回帰分析（simple regression analysis） 説明変数が1つ $latex y = a + bx$ など $latex y = ax^b$ など
重回帰分析（multiple regression analysis） 説明変数が2つ以上 $latex y = a + bx_1 + cx_2$ など $latex y = ax_1^{x_2}$ など
ここでは特に、説明変数が1つである単回帰分析、かつ、2変数の関係を1次式で表現する線形回帰、つまり、$latex y = a + bx$と表す線形単回帰分析（simple linear regeression）について扱います。

線形単回帰分析

線形単回帰分析では2つの変数の関係は1次式で表され、図示すると直線になります。この、説明変数の値から目的変数の値を予測する直線を、単回帰直線（simple regression line）といいます。

回帰式の求め方

対になったデータ$latex x_i$と$latex y_i$（$latex i$ = 1, 2, … , $latex N$）から回帰式$latex y = a + bx$を求めるには以下の数式を用います。
  1. $latex \displaystyle b = \frac{S_{xy}}{S_x}$：回帰係数（regression coefficient）
  2. $latex a = \Bar{y} - b\Bar{x}$：切片（intercept）
  • $latex \displaystyle S_{xy} = \sum_{i = 1}^N \left( x_i - \Bar{x} \right) \left( y_i - \Bar{y} \right)$：$latex x$と$latex y$との偏差積和
  • $latex \displaystyle S_x = \sum_{i = 1}^N {\left( x_i - \Bar{x} \right)}^2$：$latex x$の偏差平方和
  • $latex \displaystyle \Bar{x} = \frac{1}{N} \sum_{i = 1}^N x_i$：$latex x_i$の平均
  • $latex \displaystyle \Bar{y} = \frac{1}{N} \sum_{i = 1}^N y_i$：$latex y_i$の平均
$latex i$番目の説明変数$latex x_i$に対する目的変数$latex y_i$とある直線$latex y = a + bx$との距離（残差）を$latex Q_i$で表すと，その二乗値は$latex Q_i^2 = {\left\{ y_i - \left( a + bx_i \right) \right\}}^2$で与えられます。この$latex Q_i^2$の合計（残差平方和） $latex \displaystyle \sum_{i = 1}^N Q_i^2$ を最小にすることによって、2変数$latex x$と$latex y$との関係を表すのに最も適した回帰式を求めることができる、とする方法を最小二乗法（least squares method）と呼びます。最小二乗法で求めた$latex a$と$latex b$が上式の切片$latex a$と回帰係数$latex b$になっています。ここで、説明変数$latex x_i$に対して求めた回帰式によって予測した目的変数の値を$latex \Hat{y}_i = a + bx_i$で表すと、切片を求める式から$latex \Hat{y}_i$の平均 $latex \displaystyle \frac{1}{N} \sum_{i = 1}^N \Hat{y}_i$ は$latex \Bar{y}$と等しくなることがわかります。 なお、$latex x$と$latex y$とをそれぞれ標準化して回帰係数$latex b$を求めると，$latex b$は相関係数rと等しくなります。

回帰式の吟味

回帰式によって予測した$latex y$の値と、実際の$latex y$の値との差を残差（residual）といいます。この残差を用いて、求めた回帰係数$latex b$が予測に役立っているかを検定することができます。

仮説の設定

  • 帰無仮説H0：回帰係数$latex b$は予測に役立っていない（極端に言えば$latex b$ = 0）
  • 対立仮説H1：回帰係数$latex b$は予測に役立っている（$latex b$ ≠ 0）

有意水準αの設定

有意水準α = 0.05とします。

検定統計量$latex F$値の算出

今回、検定統計量$latex F$値は以下の数式で求めることができます。 $latex \displaystyle F = \frac{V_{\Hat{y}}}{V_Q}$
  • $latex \displaystyle V_{\Hat{y}} = \frac{1}{1} \sum_{i = 1}^N {\left( \Hat{y}_i - \Bar{y} \right)}^2$：予測値$latex \Hat{y}$の不偏分散（自由度1）
  • $latex \displaystyle V_Q = \frac{1}{N - 2} \sum_{i = 1} Q_i^2$：残差の不偏分散（自由度$latex N - 2$）
  • $latex \displaystyle \Bar{y} = \frac{1}{N} \sum_{i = 1}^N y_i = \frac{1}{N} \sum_{i = 1}^N \Hat{y}_i$：$latex y$の平均、予測値$latex \Hat{y}$の平均
残差$latex Q_i$は目的変数の値$latex y_i$と回帰式によって予測した値$latex \Hat{y}_i$とを用いて$latex Q_i = y_i - \Hat{y}_i$と表すことができます。これを$latex y_i = \Hat{y}_i + Q_i$と変形し、左辺から$latex y$の平均$latex \Bar{y}$を、右辺から$latex \Hat{y}_i$の平均$latex \Bar{y}$を、それぞれ引くと$latex y_i - \Bar{y} = \left( \Hat{y}_i - \Bar{y} \right) + Q_i$となります。これは、左辺の目的変数$latex y$の偏差$latex y_i - \Bar{y}$が右辺第1項の$latex x_i$への回帰で表される（予測値$latex \Hat{y}_i$の偏差で説明できる）部分$latex \Hat{y}_i - \Bar{y}$と右辺第2項の回帰で表されない部分$latex Q_i$とから成り立つことを意味します。この$latex y$の散らばり具合を不偏分散を用いて表せば、$latex V_y = V_{\Hat{y}} + V_Q$となります。
  • $latex \displaystyle V_y = \frac{1}{N - 1} \sum_{i = 1}^N {\left( y_i - \Bar{y} \right)}^2$：$latex y$の不偏分散（自由度$latex N - 1$）
検定統計量$latex F$は回帰式で説明できる部分の不偏分散と説明できない部分の不偏分散との比で、帰無仮説が成立する場合、$latex F$は自由度が(1, $latex N - 2$)のF分布に従うことがわかっています。

p値の算出

有意水準と比較する確率p値を計算します。p値は、自由度が(1, $latex N - 2$)のF分布において$latex F$値より大きい値が発生する確率です。帰無仮説が成立する場合、$latex V_{\Hat{y}}$が$latex V_Q$より小さくなるため、結果として$latex F$値は小さくなり、p値は大きくなります。逆に帰無仮説が成立しない場合は、$latex F$値は大きいためp値は小さくなります。

判定

F分布はこのようなグラフを描きます。このグラフでp値を表す面積が有意水準の確率を表す面積より大きいか小さいかで帰無仮説の受容か棄却かを決定します。なお、図のとおり、自由度によって、このF分布のグラフは変わります。
p値 ≦ 有意水準α
帰無仮説H0を棄却する
p値 > 有意水準α
帰無仮説H0を受容する

Excelの操作

ここまで検定の理論について説明してきましたが、Excelを用いて検定を行ってみましょう。練習問題で用いたファイルに追記します。
  1. 検定用の項目を入力し、既知のデータを入力します。
  2. 予測値$latex \Hat{y}_i = a + bx_i$を求めるために、以下のように入力し、"M2"セルを選択してオートフィルで他のセルも入力します。
    • "M2"：=K$4+K$3*B2（予測値$latex \Hat{y}_i$）
  3. 検定用のデータを算出するため、以下のように入力します。
    • "P2"：=DEVSQ(M2:M31)（予測値$latex \Hat{y}$の偏差平方和）
    • "P3"：=SUMXMY2(A2:A31,M2:M31)（残差平方和）
    • "P4"：=COUNTA(A2:A31)-2（残差平方和の自由度）
    • "P5"：=P2/(P3/P4)（検定統計量$latex F$値）
    • "P6"：=F.DIST.RT(P5,1,P4)（p値）
    SUMXMY2(配列1,配列2)は配列1と配列2の対応する要素の差の平方和を求めます（SUMXMY2関数）。
なお、FORECAST関数を用いて、"M2"を=FORECAST(B2,A$2:A$31,B$2:B$31)と、入力データから直接求めることもできます。

決定係数

回帰式は統計的に意味があることはわかりましたが、説明変数$latex x$は目的変数$latex y$の変動をどれくらい説明できるのでしょうか。 検定統計量$latex F$値の算出で述べたとおり、左辺の目的変数$latex y$の偏差$latex y_i - \Bar{y}$が右辺第1項の$latex x_i$への回帰で表される（予測値$latex \Hat{y}_i$の偏差で説明できる）部分$latex \Hat{y}_i - \Bar{y}$と右辺第2項の回帰で表されない部分$latex Q_i$とから成り立っています。そこで、$latex x_i$への回帰で表される部分$latex \Hat{y}_i - \Bar{y}$が$latex y$の散らばり具合$latex y_i - \Bar{y}$をどの程度表しているかを比で表します。回帰で表される部分に対する偏差平方和 $latex \displaystyle S_{\Hat{y}} = \sum_{i = 1}^N {\left( \Hat{y}_i - \Bar{y} \right)}^2$ と目的変数$latex y$の偏差平方和 $latex \displaystyle S_y = \sum_{i = 1}^N {\left( y_i - \Bar{y} \right)}^2$ とを用いて比を取った $latex \displaystyle R^2 = \frac{S_{\Hat{y}}}{S_y}$ を考えます。 この$latex R^2$は決定係数（coefficient of determination）と呼ばれ、0 ≤ $latex R^2$ ≤ 1です。予測値$latex \Hat{y}_i$の散らばり具合が大きければ$latex S_{\Hat{y}}$が大きくなり、$latex R^2$は大きくなるので、$latex R^2$が1に近ければ予測値の散らばり具合は目的変数$latex y$の散らばり具合に近く、説明変数$latex x$が目的変数$latex y$の変動を説明できていることになります。なお、単回帰分析では決定係数と相関係数rの二乗値とが等しい、すなわち、$latex R^2$ = r2です。 Excelで実際に求めてみましょう。
  • "P9"：=P2/DEVSQ(A2:A31)（決定係数$latex R^2$）
$latex R^2$ = 0.952494102が求まりました。求めた回帰式はよく説明できていることがわかりました。これは散布図で回帰直線が多くのデータ点と接していることに現れています。
